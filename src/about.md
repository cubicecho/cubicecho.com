Cubic Echo is a small open source software outfit. The projects shipped under
this name range from mobile and web apps to embedded tooling and developer
libraries — anything worth open-sourcing.

For now, Cubic Echo is run by a single developer. The work is unified by a
preference for small, sharp tools, public source, and projects that scratch a
real itch.

Source for everything lives on [GitHub]({{ site.github }}).
