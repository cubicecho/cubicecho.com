Cubic Echo Notes is a local-first Markdown notes application with web, server,
and Electron builds, backed by a GraphQL API and Postgres. Write in Markdown,
keep everything on your own machine, and reach the same notes from the browser
or a desktop app.

It's an early-stage monorepo (app, server, db, and Electron workspaces) that
also dogfoods Cubic Echo's own GraphQL tooling. Expect rough edges while it
takes shape.

Stack: TypeScript monorepo — React app, Apollo/GraphQL server, Drizzle +
Postgres, Electron.
